# functions

## Development

```
cd functions
yarn
```

### emulate locally (HTTP function only)
```
yarn serve
```

### emulate locally
```
yarn shell
```

## CLI

### Update Groups
```
node cli.js group
```

### Update Members
```
node cli.js member
```

### Update Schedules
```
node cli.js schedule -g <group-id> [--force]
```
